# Casio-CG100_Python-Games
A collection of python games, designed to work on the Casio CG100, fx-1AU Graph and Casio Graph Math+
(All files **should** also work with Casio CG50)
Some games have been added to this collection from other repos and some may have been modified in order to have a complete library

All games can be simply downloaded to calculator using a simple usb c data cable and following standard casio instructions

**Games:** </p>
- Noughts and Crosses - Created by Chiron8 <br>
- Simple Chrome Dino game clone <br>
- Flappy Bird <br>
- Minesweeper (OK - Flag, EXE - Uncover) <br>
- Pacman (Press exe to run) <br>
- Snake <br>
- Space Invaders (Arrows - Move, EXE - shoot) <br>
- Tetris (Original Full Screen Size - Slower due to performance scaling) <br>
- Tetris2 (Smaller size - Faster Performance)
